# Release Notes - v1.2.3

## Highlights
- Added PDF export for analysis views.
- Enhanced anomaly threshold configurator with conditional ignore/require rules.
- Added PSI HD and PSI Industrial profiles; removed legacy profiles.
- Restored RAW tab to maintain header layout.

## Improvements
- Application-aware profile grouping in the admin UI.
- Anomaly threshold rules now support parameter/unit selection.
- Import button returns to upload screen and renamed to Import New Files.
- B-Plot alerts on Overview moved below the chart for better balance.

## Notes
- Threshold configurator currently applies to B-Plot anomalies; ECM alerts remain unchanged.
